# Security Policy

## Supported Versions

We take the security of SHS Player very seriously. Below are the versions currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.15.x | :white_check_mark: |
| < v0.15 | :x:                |

## Reporting a Vulnerability

If you discover any security vulnerability in SHS Player, please DO NOT report it via public GitHub issues. 

Instead, please send an email directly to the developer at: **thejddev.official@gmail.com**

We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress. If you have not received a reply within 48 hours, please reach out via Telegram @aamoviesadmin.
